# Style Eden Blueprints

This repository serves as a collection of high-quality **Blueprints** to standardize software development practices across teams and AI agents.

## Purpose

The primary goal of these blueprints is to enforce strict standards for API contracts, lifecycle management, and core architectures to prevent drift or hallucination by AI coding assistants. By defining strict structural rules and boundaries, teams can ensure consistent quality in every generated microservice or frontend application.

## Available Blueprints

*   **`webapi`**: A platform-agnostic blueprint establishing rigorous API contract standards and payload schemas, such as standardized data/error/meta envelopes.
*   **`web-micronaut-solidstart`**: A full-stack, language-specific blueprint defining rigorous guardrails, patterns, and anti-patterns for Java 21 Micronaut backends and TypeScript SolidStart frontends.
